Programming in ANSI C
E.Balagurusamy

The book Programming in ANSI C has been developed specifically to meet the need of students. It explains concepts from elementary to advanced levels with focus on programming preliminaries. The book gives an exhaustive coverage on Arrays,Strings,Functions,pointers and Data Structures.Numerous solved examples make the content more relevant and improve learning outcomes.This book is useful for both beginners or seasoned professional.The answers to all C questions can be found in this book.
